Curriculum Overview

Weeks 1–4

QA Fundamentals & Manual Testing

+
  • Understand the software development lifecycle and QA’s role in it
  • Differentiate between testing types and levels
  • Design, execute, and manage manual test cases
  • Report bugs with clarity and reproducibility
  • Demonstrate traceability from requirements to bugs
Weeks 5–10

Web & Mobile Test Automation

+
  • Build a basic test automation framework with Selenium and C#
  • Automate functional test cases using DOM locators
  • Apply Page Object Model and maintain scripts efficiently
  • Configure Appium and run tests on real devices or emulators
  • Extend automation skills to mobile testing scenarios
Weeks 11–14

API Testing + Databases

+
  • Write and run SQL queries to validate data
  • Test REST API endpoints for functionality and accuracy
  • Create Postman collections, use environments, and automate assertions
Weeks 15–17

Performance Testing

+
  • Understand performance testing methodologies
  • Create JMeter scripts for load and stress testing
  • Generate realistic load scenarios using OctoPerf
  • Analyze and interpret performance metrics
  • Provide recommendations based on results
Weeks 18–20

Capstone Project & Career Preparation

+
  • Build your QA portfolio and résumé
  • Optimize your LinkedIn and professional profiles
  • Prepare for QA interviews and mock sessions
  • Learn to present your work as an intern
Months 6–8

Internship / First QA Job Experience

+
  • Apply your QA skills in a real-world environment
  • Collaborate with engineering teams and follow agile workflows
  • Deliver quality test documentation and reporting
  • Receive mentorship and performance feedback from senior QA professionals
