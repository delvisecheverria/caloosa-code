QA Engineer Curriculum

A complete 5-month learning experience followed by a 3-month real QA internship. Learn, build, and work like a professional QA Engineer.

Weeks 1–4 QA Fundamentals & Manual Testing
+

Goals

  • Understand the software development lifecycle and QA’s role in it
  • Differentiate between testing types and levels
  • Design, execute, and manage manual test cases
  • Report bugs with clarity and reproducibility
  • Demonstrate traceability from requirements to bugs

Content

  • Introduction to the Software Development Life Cycle (SDLC)
  • Types and levels of testing, QA roles
  • Test case design and execution
  • Test case management using AIO Test / TestRail
  • Bug reporting and management in Jira
  • Hands-on practice with a sample web application and full traceability
Weeks 5–10 Web & Mobile Test Automation
+

Goals

  • Build a basic test automation framework using Selenium and C#
  • Automate functional test cases with DOM locators and assertions
  • Apply the Page Object Model for scalability and maintenance
  • Integrate tests with BrowserStack and version control via GitHub
  • Extend automation to mobile applications using Appium

Content

  • Web Automation (Weeks 5–7):
    • Selenium + C# fundamentals
    • Test framework structure using NUnit or MSTest
    • DOM interaction, locators, and Page Object Model
    • Integration with BrowserStack and GitHub
  • Mobile Automation (Weeks 8–10):
    • Mobile testing fundamentals (iOS / Android)
    • Appium setup for local and emulator testing
    • Test framework structure for mobile automation
    • Page Object Model and locator strategies for mobile apps
Weeks 11–14 API Testing + Databases
+

Goals

  • Write and run SQL queries for validation
  • Test REST API endpoints for accuracy
  • Create Postman collections and automate assertions
  • Validate backend data consistency

Content

  • SQL fundamentals: SELECT, INSERT, UPDATE, DELETE
  • Test data validation using DBeaver
  • REST API fundamentals: JSON, HTTP methods
  • API automation with Postman and Newman
Weeks 15–17 Performance Testing
+

Goals

  • Understand performance testing methodologies
  • Create and execute load and stress tests
  • Generate realistic load scenarios using OctoPerf
  • Analyze reports and optimize configurations

Content

  • Types of load, stress, and endurance tests
  • Scripting with Apache JMeter
  • Load generation with OctoPerf
  • Result analysis and performance monitoring
Weeks 18–20 Capstone Project & Career Preparation
+

Goals

  • Apply all QA skills to a real-world project
  • Collaborate in agile QA teams
  • Develop professional QA documentation
  • Prepare for job interviews and build your portfolio

Content

  • Comprehensive QA project from planning to execution
  • Jira + GitHub workflow integration
  • Mock interviews and resume preparation
  • Portfolio and presentation showcase
Months 6–8 Internship / Real-World QA Project
+

Goals

  • Gain hands-on QA experience in a company setting
  • Apply QA methodologies across multiple testing areas
  • Collaborate with real QA and dev teams
  • Build confidence through mentorship
  • Earn your first professional QA experience

Content

  • 3-month guided internship with a Caloosa Code partner
  • Participation in live QA projects (manual, automation, API)
  • Mentorship, team collaboration, and weekly feedback
  • End-of-internship report and performance evaluation
  • First QA job experience for your resume
