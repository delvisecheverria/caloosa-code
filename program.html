▶ WEEKS 1 – 4: QA FUNDAMENTALS & MANUAL TESTING

Content:

  • Introduction to the Software Development Life Cycle (SDLC)
  • Types and levels of testing, QA roles
  • Test case design and execution
  • Test case management using AIO Test / TestRail
  • Bug reporting and management in Jira
  • Hands-on practice with a sample web application and full traceability

Objectives:

  • Understand the software development lifecycle and QA’s role in it
  • Differentiate between testing types and levels
  • Design, execute, and manage manual test cases
  • Report bugs with clarity and reproducibility
  • Demonstrate traceability from requirements to bugs
▶ WEEKS 5 – 10: WEB & MOBILE TEST AUTOMATION

Content:

  • Web Automation (Weeks 5 – 7): Selenium + C#, NUnit or MSTest, DOM locators, Page Object Model, BrowserStack, GitHub
  • Mobile Automation (Weeks 8 – 10): Appium, iOS / Android setup, framework creation, locators for mobile apps

Objectives:

  • Build a test automation framework using Selenium + C#
  • Automate test cases with DOM locators
  • Apply Page Object Model and maintain reusable scripts
  • Configure Appium for device and emulator testing
  • Integrate tests with BrowserStack and GitHub
▶ WEEKS 11 – 14: API TESTING + DATABASES

Content:

  • SQL fundamentals: SELECT, INSERT, UPDATE, DELETE
  • Data validation using DBeaver
  • REST API testing with Postman — JSON, HTTP methods, body, and assertions

Objectives:

  • Write and execute SQL queries for validation
  • Test API endpoints and automate validations
  • Use Postman for REST API automation
▶ WEEKS 15 – 17: PERFORMANCE TESTING

Content:

  • Understand load, stress, and endurance tests
  • Scripting and analysis with JMeter
  • Load generation using OctoPerf
  • Performance tracking and monitoring

Objectives:

  • Build test plans for different performance scenarios
  • Run JMeter scripts and analyze results
  • Interpret metrics and create optimization reports
▶ MONTH 6 – 8: INTERNSHIP / REAL-WORLD QA PROJECT

100% job placement support for graduates, plus real-world QA experience with live projects.
