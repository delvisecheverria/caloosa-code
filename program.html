QA Engineer Curriculum

A 5-month intensive program designed to prepare you for your first QA job through hands-on learning, real projects, and mentorship.

Weeks 1–4

QA Fundamentals & Manual Testing

  • Understand SDLC and QA’s role in it
  • Differentiate between testing types and levels
  • Design and execute manual test cases
  • Report bugs with clarity and reproducibility
  • Use Jira, TestRail, and AIO Test effectively
Weeks 5–10

Web & Mobile Test Automation

  • Build frameworks with Selenium & C#
  • Apply Page Object Model for test maintenance
  • Run Appium tests on real devices
  • Version control with GitHub
Weeks 11–14

API Testing + Databases

  • Validate REST API endpoints with Postman
  • Use environments and automate assertions
  • Run SQL queries for data verification
  • Tools: Postman, DBeaver
Weeks 15–17

Performance Testing

  • Understand load, stress, and endurance testing
  • Create JMeter scripts for performance validation
  • Analyze test results and performance metrics
Weeks 18–20

Capstone Project & Career Preparation

  • Build your QA portfolio
  • Prepare for interviews and mock sessions
  • Present your final QA project
  • Receive personalized feedback
