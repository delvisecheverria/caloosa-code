Weeks 1–4: QA Fundamentals & Manual Testing

Content:

  • Introduction to the SDLC and QA roles
  • Types and levels of testing
  • Test case design and execution
  • Test management in AIO Test / TestRail
  • Bug tracking in Jira

Objectives:

  • Understand the QA role in SDLC
  • Design and execute manual test cases
  • Report and track bugs effectively

Weeks 5–10: Web & Mobile Test Automation

Content:

  • Selenium + C# automation
  • DOM locators & Page Object Model
  • BrowserStack integration
  • Mobile testing with Appium (iOS/Android)

Objectives:

  • Build automation frameworks
  • Automate and reuse scripts
  • Integrate with GitHub and CI

Weeks 11–14: API Testing + Databases

Content:

  • SQL basics: SELECT, INSERT, UPDATE, DELETE
  • Data validation with DBeaver
  • REST API testing with Postman

Objectives:

  • Validate data through SQL
  • Test and automate REST APIs

Weeks 15–17: Performance Testing

Content:

  • Understand load, stress, and endurance tests
  • JMeter scripting and analysis
  • Load generation with OctoPerf

Objectives:

  • Design test plans for performance
  • Analyze results and optimize performance

Months 6–8: Internship / Real-World Project

Content:

Objectives:

  • Build a professional QA portfolio
